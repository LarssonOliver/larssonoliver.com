# Oliver's Portfolio website.

Deployed website can be found at [larssonoliver.com](https://larssonoliver.com).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
